## üìã Next Steps (User Action Required)

1. **Install dependencies**:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # Windows
   pip install -r requirements.txt
   ```

2. **Initialize DVC**:
   ```bash
   dvc init
   mkdir -p ../dvc-storage
   dvc remote add -d localstorage ../dvc-storage
   ```

3. **Add GitHub remote** (if using GitHub):
   ```bash
   git remote add origin <your-github-repo-url>
   ```

4. **Add data file**:
   ```bash
   # Place insurance.csv in data/raw/
   dvc add data/raw/insurance.csv
   git add data/raw/insurance.csv.dvc .gitignore
   git commit -m "task-2: add raw data tracked by dvc"
   dvc push
   ```

5. **Run pipeline**:
   ```bash
   dvc repro
   ```
